## <picture><img src="https://github.com/7oSkaaa/7oSkaaa/blob/main/Images/about_me.gif?raw=true" width="30px"></picture> About Me

I'm a **Multiplatform Software Development** student (TSU), passionate about building things for the web and growing as a developer every single day. I'm currently focused on **Full Stack development** — strengthening my backend skills with **Laravel** and leveling up on the frontend with **React** and **TypeScript**.

I believe the best way to learn is by doing, so I'm always working on personal projects and challenging myself with new technologies. My long-term goal is to become a **software architect** — but for now, I'm enjoying every step of the journey as a junior dev hungry to grow. 🚀
